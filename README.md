## 💎 The Vision
**Aspen** is a glassmorphism-inspired Studio Suite built for professionals who value clean design and functional efficiency. It transforms the complex task of PDF manipulation into a fluid, mobile-first experience.

## 🚀 Core Features
* **Text-to-PDF Engine**: Draft high-fidelity documents with a distraction-free interface.
* **Media Integration**: Convert visual assets into portable document formats instantly.
* **Dynamic Overlays**: Professional-grade PDF editing with custom text positioning.
* **Live Web Capture**: Snap any URL and generate a documented version in seconds.
* **Particle UI**: A dynamic, interactive background system for an immersive entry experience.

## 🛠️ Tech Stack
* **Framework**: Tailwind CSS (Utility-first styling)
* **Logic**: Vanilla JavaScript (ES6+)
* **PDF Core**: `jsPDF` & `pdf-lib`
* **Icons**: Lucide & Custom SVG Paths
* **Typography**: Plus Jakarta Sans
